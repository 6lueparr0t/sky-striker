# Sky Striker — 개발 기록

세로 스크롤 비행기 슈팅 게임. Neutralinojs 데스크톱 앱.

## 왜 이 스택인가 (배경)

- Electron은 용량(크롬 통째 번들 → 100MB+)이 부담.
- → **Neutralinojs** 선택: OS 웹뷰 사용(앱 2~5MB), 웹기술(HTML/CSS/JS) 그대로, CLI는 npm·바이너리는 github에서 받아 **폐쇄망에서 문제없이 빌드됨**. Rust/crates.io 안 건드림.

## 기술 구성

- **프레임워크/엔진 없음.** 순수 바닐라 JS + HTML5 Canvas 2D.
- 게임 루프: `requestAnimationFrame` + delta-time.
- 효과음: **WebAudio로 실시간 합성** (오디오 파일 0개 → 폐쇄망 OK).
- **BGM**: WebAudio 합성 칩튠 루프(A단조 펜타토닉, 16스텝 시퀀서). 오디오 파일 없음. 보스전엔 템포↑·음량↑. `updateBGM()`/`scheduleBgmStep()`. 첫 키 입력 때 오디오 언락되며 시작.
- 아트: 전부 캔버스 도형으로 그림 (외부 이미지 0개).
- 하이스코어·클리어 횟수: `localStorage` **+ `Neutralino.storage`(.storage/) 이중 저장**(웹뷰 localStorage가 세션 간 안 남는 경우 대비). `persistSet()`/`persistLoad()`. `nativeAllowList`에 `storage.*` 필요.

```
sky-striker/
├── neutralino.config.json   # 창 500x780, 제목 Sky Striker, enableInspector:false
└── resources/
    ├── index.html           # 캔버스만
    ├── styles.css           # 다크 배경 + 네온 글로우
    └── js/
        ├── main.js          # Neutralino 부트스트랩(init + windowClose)
        └── game.js          # 게임 전체 로직
```

## 실행 / 빌드

```bash
neu run                 # 개발 실행
neu build               # dist/sky-striker/ 에 배포용 바이너리
neu build --release     # 배포용 zip까지
```

배포본: `dist/sky-striker/` 의 **`sky-striker-win_x64.exe` + `resources.neu`** 두 파일 세트면 실행됨(~2MB). 나머지 플랫폼 바이너리는 삭제해도 무방.
> `neu run`을 백그라운드로 띄우면 창 닫을 때 CLI가 exit 255를 뱉지만 앱 자체는 정상 종료(success code 0). 무시해도 됨.
> 실행 로그의 `favicon.ico` / `com.chrome.devtools.json` 404 ERROR도 무해(웹뷰 기본 요청).

## 조작

| 키 | 기능 |
|----|------|
| 방향키 | 이동 (WASD 아님) |
| Z | 발사 / (메뉴·엔딩 화면에서도 시작 가능) |
| X | 패링 |
| C | 탄막 소거 (게이지 가득 찼을 때) |
| Enter | 시작 · 스토리 넘김 |
| ↑ / ↓ | (메뉴) 게임 시작 / 무한 모드 선택 |
| ← / ↑ | 스토리 이전 페이지 |
| P / ESC | 일시정지 메뉴 |
| Alt+Enter (Mac ⌘+Enter) | 전체화면 토글 |
| (자동) | 유도미사일 — 아이템 먹으면 자동 발사 |

> 창 크기 조절 가능(`resizable`), 기본 640x960. 캔버스는 내부 480x720 고정 해상도를 창 크기에 맞춰 CSS 스케일(2:3 유지). `game.js`의 `fitCanvas()`/`toggleFullscreen()`.
> 전체화면 토글은 **Alt+Enter / Mac ⌘+Enter**(keydown에서 직접 처리). `Neutralino.window.setFullScreen()`을 쓰므로 `neutralino.config.json`의 `nativeAllowList`에 **`window.*`** 필수(없으면 조용히 차단됨 — 예전 F키가 안 먹던 원인).

## 구현된 기능

### 스테이지

- 총 5스테이지. 스테이지별 quota(스폰 수)·스폰속도·체력배수·적 풀이 다름. 스테이지 1은 완화(느린 스폰, 적 발사 빈도↓, 캐리어 등장률↑).
- **볼륨 상향**: 스테이지 길이(quota) 대폭 상향(160/224/256/304). 중간보스 체력 ×4, 최종보스 ×16(원본 대비).
- **스폰 편대 패턴**: 가로 일렬 / V자 / 세로 스트림 / 단독.
- **중간 보스**: 스테이지 2, 4에서 일반 적 소탕 후 등장. **각각 다른 이미지** — 2는 아폴로형 순양함, 4는 **아주 큰 우주정거장**(회전 도킹 링·다단 거주 드럼·대형 태양광 날개, 체력 1.6배). 자체 체력바, 2패턴. 잡아야 진행.
- **최종 보스**: 스테이지 5 (아래 참고).
- **무한 모드**(메뉴에서 **↑↓로 선택 후 Enter**): 스테이지 1~5가 계속 반복. 최종 보스 격파 시 스테이지 1로 루프(플레이어 파워·목숨·조력자 유지). 루프마다 적·보스 체력 상승(`loopScale()` = 1 + 루프×0.35). HUD에 `∞ LOOP n · s/5` 표시.

### 적 종류 (`ETYPE`) — 인류 우주선 모양(정체 복선)

| 타입 | 특징 |
|------|------|
| basic | 로켓, 직진 |
| zigzag | 위성, 좌우 지그재그 |
| shooter | 위성, 체력3, 5발 조준 산탄 |
| fast | 소형 로켓, 초고속 돌격 |
| tank | 스테이션 모듈, 체력10, 방사형 탄막, 체력바 |

### 최종 보스 (배틀크루저 / 사이버펑크)

- 체력 7680. 디자인은 주인공 인터셉터의 크고 사이버펑크한 버전(네온 엣지, 페이즈2 핫핑크 전환, 4엔진, 윙팁 포드).
- **분노 스케일링**: 체력이 깎일수록 탄 수↑·발사 간격↓·탄속↑.
- **패턴 11종**: 부채꼴 / 조준 산탄 / 회전 나선 / 역이중 나선 / 랜덤 샷건 / 틈 있는 탄막벽 / **폭죽 탄(날아가다 링으로 터짐)** / **초고속 니들** / **유도미사일 스웜** / **초고속 레일탄(빨간 장탄 · 간신히 패링 가능)** / **차징 레이저(2~4빔)**.
- **패링 불가 탄**: 폭죽 탄(미사일)과 유도미사일 스웜은 `noParry` 플래그로 **X 패링으로 못 막음**(피하거나 C 탄막소거로만 처리). 초고속 레일탄은 패링 가능하지만 속도가 빨라 타이밍이 빡셈.

### 플레이어 능력

- **발사(Z)**: 파워 **1~5단**. 레벨업 시 탄 수/스프레드 증가(5단=7way). 탄 데미지도 미세 상승 **1 / 1.2 / 1.5 / 2 / 2.5**(`POWER_DMG`).
- **패링(X)** — 젤다식: 0.5초 방어창 / **쿨 1.5초**. 적 탄을 초고속(1300) **유도 반사**(3배 데미지). 막는 순간 창이 리셋돼 **연속 패링** 가능.
- **피격 히트박스**: 스프라이트(30×34)보다 훨씬 작은 코어(12×14)로만 데미지 판정 → 탄 회피가 관대함. 아이템 획득은 스프라이트 크기 그대로.
- **탄막 소거(C)**: 패링 반사마다 게이지 충전(반사당 +5 → **가득 채우려면 20회**), 가득 차면 C로 **화면 탄막 전체 제거 + 화면 플래시**.
- **유도미사일**: `M` 아이템 먹으면 자동 발사. **레벨 1~5**, 0.85초 간격, 속도 810. 살보는 **좌우 2발씩 페어**로, 레벨 수만큼(레벨=페어 수) 0.14초 간격 순차 발사.
- **적 강화**: 전 적 체력 ×1.5(`ENEMY_HP_MUL`), 발사 간격 단축·탄 수 증가(basic/zigzag는 조준 2연발, shooter 5발, tank 18발 방사) — 패링 견제.

### 아이템 — **캐리어 적만** 드랍

- 편대 중 **금색으로 반짝이는 캐리어 적**을 잡을 때만 드랍(스테이지1은 등장률↑).
- **P** 파워업 / **M** 미사일 레벨업 / **♥** 목숨+1.
- **스테이지당 드랍 예산**(`resetStageItems`): P·M 각 **1~2개**, ♥ **정확히 1개 보장**(캐리어에서 하트 우선 드랍, 못 나오면 클리어 직전 플레이어 근처에 안전 드랍).

### 스토리 / 엔딩

- **인트로** 컷신 → 플레이. 스토리: 지구를 침공한 함대는 사실 **과거에 지구를 떠난 인류**(블랙홀/시간 왜곡으로 서로를 못 알아본 비극).
- **1회차 보스 격파**: 진실 컷신 → 엔딩 → 메뉴 (재도전 힌트).
- **2회차 보스 격파**: **히든 스테이지** 개방.
- 컷신 시스템: 아트(지구/함대/블랙홀/해치/빛/하트) + 텍스트 + 페이지 점. Enter 다음, ←/↑ 이전.

### 히든 스테이지 (블랙홀 구출)

- 사격 없음. **블랙홀 인력**(가까울수록 강함, 중심 삼켜지면 배드엔딩)을 버티며 이동.
- **우주선 1대 + 우주인 9명 전원 구출**하면 진엔딩. 우주인은 블랙홀로 서서히 끌려감(못 구하고 삼켜지면 배드엔딩).
- 블랙홀: 정지 타원 어크리션 디스크 + 물질 공전 + 검은 코어 + 광자 링.
- **진엔딩**: 해치 열림 → 빛(새로운 시작) → 하트 + 시적 문구 → "두 인류는 마침내 같은 지구 아래 선다" → TRUE END.

### 이스터에그 / 기타

- **디스코 모드 상시 ON**: 무지개 별 배경 + 함선 무지개 트레일.
- **클로이 UFO → 아군 조력자**: **스테이지마다 50% 확률**로 하트 단 보라색 UFO(체력 3, HP바)가 등장. 격추하면 조력자로 합류(**최대 3기**, 영구) → 자율 비행하며 가까운 적에 유도탄 발사. (패링 없음 · 점수 보상 아님)
- **디버그 단축키(의도적 유지)**: `9`=최종보스, `8`=엔딩 컷신, `7`=히든 스테이지 바로 진입.
- 목숨 3(최대 5), 피격 시 무적 1.6초(**파워는 유지 — 감소 없음**).
- 일시정지 메뉴(재개/메뉴로/종료). 별 배경 스크롤, 폭발/색종이 파티클, 화면 흔들림·플래시.

## 튜닝 포인트 (game.js 내 위치)

- `STAGES` 배열: quota·spawn·hpMul·pool·midboss → 스테이지 난이도
- `ETYPE` 객체: 적별 hp·spd·fire·every → 적 강함
- `updateBoss()`: 보스 패턴/분노 스케일링/레이저 빔 수
- `activateParry()`: 패링 창(0.5)/쿨(1.5)
- `updateMissiles()`: 미사일 속도(810)·선회(10) / 순차 발사 간격은 `update()`의 `missileGap`(0.13)·`missileTimer`(0.85)
- `updateChloe()`: 클로이 조력자 이동·유도탄·패링
- `spawnBoss()`(hp 7680) / `spawnMidboss()`(kind: apollo/station, station은 stageIndex≥3, 체력 ×4)
- `fitCanvas()`(창 맞춤 스케일) / `toggleFullscreen()`(F키) / `textBg()`·`plated()`(글자 뒤 어두운 배경)
- `startHidden()`: 구출 대상 수(우주선1/우주인9), `updateHidden()`: 블랙홀 인력(pull)·우주인 끌림
